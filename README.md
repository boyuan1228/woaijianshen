## 项目简介

Meet Prep App 是一个专注于力量训练的 Web 应用，旨在帮助教练和训练者更高效地制定、管理和跟踪训练计划。

项目最初源于日常带学员过程中对训练计划管理的需求，希望将繁琐的 Excel、微信聊天记录和纸质训练日志整合到一个平台中，实现更加便捷、直观的训练管理体验。

目前项目仍在持续开发和优化中，欢迎大家提出建议与反馈。

---

## 功能特点

* 📋 学员档案管理
* 📈 周期训练计划生成
* 🏋 多种训练体系支持
* 📊 RPE / 百分比自动计算
* 📉 MEV / MRV 训练建议
* 💪 三项专项辅助动作推荐
* 🔄 渐进超负荷自动安排
* 📄 PDF 导出
* 📱 响应式网页设计
* 🌐 支持电脑与移动端访问

---

## 支持的训练体系（持续更新）

目前已支持或正在开发：

* JTS 风格周期
* SSTT 风格训练
* DUP Daily Undulating Periodization
* Upper / Lower
* Push Pull Legs (PPL)
* 力量举专项训练
* 肌肥大周期
* 自定义训练模板

## 项目目标

* 用户登录

* PR 历史记录
